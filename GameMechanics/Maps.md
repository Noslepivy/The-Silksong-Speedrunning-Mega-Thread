# Maps

A set of maps containing different sets of information useful to speedrunning.

---

### [MapGenie Interactive Map](https://mapgenie.io/hollow-knight-silksong/maps/pharloom)
MapGenie generally creates high quality interactive maps, containing A LOT of useful information, and trackable categories, filterable in an easy to use UI.  

![MapGenie](/Media/Screenshots/Maps/MapGenie.png)

---

### [Map of Room Codes](https://silksong-completion.info/map_of_names/)
A map complete with the roomcodes and screenshots of each room, helpful for warping with some mods, keycoding information, and searching for specific areas and information.  

![Roomcodes1](/Media/Screenshots/Maps/Roomcodes1.png)  
![Roomcodes2](/Media/Screenshots/Maps/Roomcodes2.png)  

---

### [5000 Screenshot Interactive Map](https://scripterswar.com/silksong/map)
A Map that's been put together with many, many, many, many... many in game screenshots, to create a comrehensive easily viewably interactive map.

![5000Map1](/Media/Screenshots/Maps/5000Map1.png)  
![5000Map2](/Media/Screenshots/Maps/5000Map2.png)
